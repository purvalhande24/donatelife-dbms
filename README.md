# 💝 Donation System

A desktop-based Donation Management System built in Java with MySQL database integration.

## 📋 About

This application helps manage and track donations, donors, and records efficiently.
Built as a Java desktop application using NetBeans IDE.

## 🛠️ Tech Stack

- **Language:** Java
- **Database:** MySQL
- **IDE:** NetBeans
- **DB Connector:** MySQL Connector/J
- **Build Tool:** Apache Ant

## ✨ Features

- Add and manage donor information
- Track donation records
- MySQL database integration for persistent data storage
- Simple and clean desktop UI

## ⚙️ How to Run

1. Install [NetBeans IDE](https://netbeans.apache.org/)
2. Install [MySQL](https://www.mysql.com/) and set up the database
3. Download [MySQL Connector/J](https://dev.mysql.com/downloads/connector/j/) and add it to the project classpath
4. Clone or download this repository
5. Open the project in NetBeans
6. Run the project using the **Run** button
